Formulae

Periastron Advance

$$\dot{\omega} = \frac{3}{1-e^2}\left(\frac{2\pi}{P_b}\right)^{5/3}\left(T_\odot\, M\right)^{2/3}$$
where $T_\odot = GM_\odot/c^3 = 4.925491\;\mu\text{s}$ and $M = m_1 + m_2$ in solar masses.

Einstein Delay Amplitude

$$\gamma = e\; T_\odot^{2/3}\left(\frac{P_b}{2\pi}\right)^{1/3}\frac{m_2\left(m_1 + 2m_2\right)}{M^{4/3}}$$

Orbital Period Derivative (GW emission)

$$\dot{P}_b = -\frac{192\pi}{5}\;T_\odot^{5/3}\left(\frac{2\pi}{P_b}\right)^{5/3} f(e)\;\frac{m_1\,m_2}{M^{1/3}}$$ $$f(e) = \frac{1 + \tfrac{73}{24}e^2 + \tfrac{37}{96}e^4}{\left(1-e^2\right)^{7/2}}$$

Merger Time (Peters 1964)

$$T_{\rm merge} = \frac{12}{19}\;\frac{c_0^{\,4}}{\beta}\int_0^{e_0}\frac{e'^{\,29/19}\left(1+\tfrac{121}{304}e'^{\,2}\right)^{1181/2299}}{\left(1-e'^{\,2}\right)^{3/2}}\,de'$$ $$\beta = \frac{64}{5}\frac{G^3\,m_1\,m_2\,M}{c^5},\qquad c_0 = \frac{a_0\left(1-e_0^2\right)}{e_0^{\,12/19}\left(1+\tfrac{121}{304}e_0^2\right)^{870/2299}}$$
Numerically integrated with trapezoidal rule ($N=2000$ steps).

Shapiro Delay

$$\Delta t_S = -2r\;\ln\!\Big(1 - s\,\sin(\omega+\nu)\Big)$$ $$r = T_\odot\,m_2,\qquad s = \sin i$$

Roemer Delay

$$\Delta t_R = x\Big[\sin\omega\,(\cos E - e) + \sqrt{1-e^2}\,\cos\omega\,\sin E\Big]$$ $$x = a_1\sin i / c \quad\text{(projected semi-major axis)}$$
